# Financial Market Forecasting & Analytics

A financial time-series analytics project using Python, yFinance, statistical modeling, and forecasting techniques to analyze and forecast stock market behavior.

## Objectives

- Fetch live market data using yFinance
- Perform financial time-series preprocessing
- Analyze trends, volatility, and seasonality
- Build ARIMA and Prophet forecasting models
- Create interactive dashboards for business intelligence

## Tech Stack

- Python
- yFinance
- Pandas
- NumPy
- Plotly
- Statsmodels
- Prophet
- Power BI

## Assets Analyzed

- AAPL (Apple)
- MSFT (Microsoft)
- SPY (S&P 500 ETF)
